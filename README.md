# Web CAD

A collaborative (soon) online 2D CAD platform using Konva.js

![](https://img.shields.io/badge/online%20collaboration-8A2BE2)

### Features

- Zooming and panning
- Snapping onto nearest point
- Ortho mode (press shift)
- Select and deselect
- Shortcuts
